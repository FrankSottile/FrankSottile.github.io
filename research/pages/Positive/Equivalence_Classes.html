Equivalence classes of chains in the Grassmannian Bruhat order

These are the maple files created by makeClasses.maple. Equivalence_Classes/Classes.n requires Equivalence_Classes/Classes.n-1 and all begin with Equivalence_Classes/Classes.1, which must be initialized, along with the directory Equivalence_Classes/.

    Each Maple-readable file defines tables Equivalence_Classes[n][k], where n is the length of the flattened words and k is the number of distinct numbers that appear in the chain. A chain of length n is stored as a list of integers [a1,b1,..., an,bn] of length 2n, where ai,bi corresponds to the transposition tai,bi.
