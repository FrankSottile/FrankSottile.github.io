Connected Dual Equivalence Graphs of chains in the Grassmannian Bruhat order of lengths up to six.

These are the maple files created by makeGraphs.maple. This maple script reads Equivalence_Classes/Classes.n, and sorts its chains into connected dual equivalence graphs consisting of k vertices in the files Graphs.n/g.k.
    Each file maple-readable file Graphs.n/g.k defines a set G whose elements are the graphs from chains of length n with k vertices. Each graph is represented by a pair [Ch,Edges], where Ch is the list of the chains in that graphs, and Edges is a set whose elements are the edges. Each edge is a triple {col,verta,vertb}, where col is the color of the edge, a number between 2 and n-1, and Ch[verta], Ch[vertb] are the vertices connected by the edge, where we have verta < vertb.
