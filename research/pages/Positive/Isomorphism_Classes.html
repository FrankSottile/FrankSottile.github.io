Representatives of Isomorphism Classes of connected Dual Equivalence Graphs of chains in the Grassmannian Bruhat order of lengths up to six.

These are the maple files created by graphIsomorphism.maple. This maple script reads each file Graphs.n/g.k, which consists of a list of all connected dual equivalence graphs consisting of k. It sorts these into isomorphism classes and writes a representative of each class in the file Automorphism_Types.n/at.k.
    Each file maple-readable file Automorphism_Types.n/at.k defines a list Isomorphism_Classes[n,k] whose members are representatives of isomorphism classes of graphs from chains of length n with k vertices. Each graph is represented by a quadruple [Des, Edges, Ch, Balls], where Des is an ordered list of descent sets, Edges is a set whose elements are the edges, each of which is a triple [edge_color, vertex_a, vertex_b], Ch is the list of the chains in that graph, and Balls is a table of the vertces and edges at a fixed distance from a distinguished vertex that is used to compute isomorphisms.
