Rosenthal, Joachim (1-NDM); Sottile, F. (3-TRNT)
Some remarks on real and complex output feedback. (English. English summary)
Systems Control Lett. 33 (1998), no. 2, 73--80.
Summary: "We provide some new necessary and sufficient conditions which guarantee arbitrary pole placement of a particular linear system over the complex numbers. We exhibit a nontrivial real linear system which is not controllable by real static output feedback and discuss a conjecture from algebraic geometry concerning the existence of real linear systems for which all static feedback laws are real."
